	1，javascript面向对象
		
			
		（1）面向对象oop（Object Oriented Programming）：把事物分解成为一个个的对象，然后对象之间分工合作。
		

		（2）面向对象和面向过程的区别：
			
			
			1，面向过程用于小项目。面向对象用于多人合作的大项目
		
		
		（3）面向对象的三大特性：
			
			
			1，封装性：把某些功能以及封装好了，调用即可执行。
			
			2，继承性：子承父业，可以继承父辈的某些功能。
			
			3，多态性：一个对象多种形态
		
		
		（4）面向对象和面向过程的优缺点：
			
			
			（1）面向过程的性能要比面向对象的性能好，步骤精密耦合。
			
			
			（2）面向过程不容易维护，不易多次使用，扩展功能差。
			

			（1）面向对象易维护，可服用，可扩展，灵活性高


			（2）面向对象性能没有面向过程稿
		
			
		
	2，ES6中的类和对象：
		
			
		（1）类和对象的感念：值一类事物，比如人就是一类，而某一个人就是对象，是一类事物的具体的表现。
			
			
		（2）类的创建：通过class来创建一个类。类的命名第一个字母必须大写。通过constructor属性可以指定类中的构造函数。需要初始化的属性都在构造函数中完成。
			
				
			格式：class Person {


				construnctor() {
				
				}

			      }
		

			
		（3）类中的方法和属性：
			
			
			（1）方法不能加function，直接使用函数名即可，但是在使用函数名定义的函数里面可以使用带有function的函数。

			
			（2）属性访问必须加上this。

			
			
		（4）this的指向：
			
			
			（1）谁调用方法或者属性，this就指向谁。
			
		
		（5）创建对象：
			
			
			（1）通过new关键字，可以创建对象。使用new 加上类名，会返回一个对象。
				
				
				
				格式： var obj = new Person()
		
			

		（6）类的继承：
			
			
			（1）通过extends关键字可以一个类继承另一个类的属性和方法
			
			
			（2）如果子类和父类都有构造函数的话，可以直接调用super()方法代表当前父类。可以调用父类的构造格式，也可以调用父类的普通的方法
			
			
		
		
		（7）三个注意点：
			
			
			（1）在ES6中类没有变量提升，所以必须先定义类，才能通过类来实例化对象
			
			
			（2）类里面的属性和方法一定要加this来使用。
			
			
		
		（8）static静态：在方法或者属性的前面加上static关键字的话，那么此时的方法和属性不能被实例化的对象调用，只能被类名来调用。
